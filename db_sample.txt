the file loopware.db contains some systems created with Making Software.

Use the commands bellow to restore to your environemnt for tests purpose.
Use postgres user to perform this tasks.

1. criar o  banco no destino:
createdb -h localhost -T template0  <db_for_test_name>

2. exportar o banco na origem (Used to generate the db dump)
pg_dump -h localhost -f making_software_sample.db -F p -U <db_user> <db_to_export>

3. importar o banco no destino:
psql -h localhost -f making_software_sample.db <db_for_test_name> <db_user>
