Making Software

Crie o seu sistema em apenas 4 passos

Primeiro passo: Cadastre-se

É muito simples, você só precisa prencher os dados abaixo

  • Nome da sua empresa
  • Endereço de e-mail válido
Ou se preferir pode efetuar o seu login através de uma rede social em apenas um clique (Facebook, Twitter, Google+)

Segundo passo: Crie o seu Sistema

Para criar o seu sistema você só precisa definir os dados abaixo:

  • Nome do sistema (Título)
  • Subtítulo para o sistema
  • Descrição completa do sistema
Essas três informações serão apresentadas na tela inicial do seu sistema.

Terceiro passo: Crie o Menu do Sistema

O sistema é organizado por sessões, ou elementos dispostos em um menu.
Cada sessão possui um formulário para o usuário preencher as informações que você deseja coletar.
Você cria as sessões do seu sistema, e define as informações abaixo:

  • Nome da sessão (Título)
  • Subtítulo da sessão
  • Descrição completa da sessão
Essas três informações serão apresentadas na tela das respectivas sessões do seu sistema.

Quarto passo: Crie os Formulários do Sistema

Cada sessão do seu sistema possui um formulário.
Você cria os formulários com os campos que desejar, e define tudo em uma interface amigável e intuitiva.
Para cada campo do formulário você define os dados abaixo:

  • Tipo do campo (texto, número, data, e-mail, checkbox, radio button etc)
  • Nome
  • Obrigatoriedade de prenchimento
Durante a criação dos campos, ou após a definição de todos os campos, você posiciona os campos na ordem desejada.

 

Pronto!!!

Em apenas quatro passos você criou o seu sistema.
Com um clique no botão "Publicar", o seu sistema será publicado automaticamente na Internet.
O sistema é automaticamente criado com todas as funcionalidades:

  • Formulários que você definiu
  • Menu com as sessões do sistema
  • Mecanismo de cadastro de usuários
  • Interface amigável para você administrar e visualizar usuários e demais informações cadastradas

     Começar!
