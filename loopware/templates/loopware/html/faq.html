FAQ

Perguntas Frequentes

  1. O meu sistema fica dentro do InteliForm?
    Não, o seu sistema é completamente independente do InteliForm. Ele utiliza um subdomínio do InteliForm (https://projetos.inteliform.com.br), por questões práticas. O seu sistema poderia estar hospedado em qualquer domínio, em qualquer servidor.

  2. Senhas Criptografadas
    O InteliForm não tem acesso a nenhuma senha dos usuários, elas são armazenadas criptografadas e não são descriptografadas em nenhum momento.

  3. HTTPS/SSL - O que é isso?
    HTTPS/SSL significa que toda a comunicação trocada entre o browser que o cliente utiliza para navegar em um sistema e o servidor que hospeda esse sistema, é criptografada. E caso alguém consiga interceptar a comunicação, não conseguirá decifra-la justamente porque ela está criptografa.
    O seu sistema utiliza uma URL segura(https://projetos/inteliform.com.br/). dessa maneira todas as informações
    que trafegam entre o seu sistema e o browser do seu usuário é criptografada e ninguem pode acessar.

  4. Usuário administrativo do seu sistema
    O admin do seu sistema é você. O seu sistema é gerado com o mesmo usuário que você utiliza no Inteliform.
    Porém, se você alterar a senha desse usuário no Inteliform, ela não será alterada no seu sistema, e vice-versa. Os dois sistemas são 100% independentes.

  5. Primeiro login no meu sistema
    Quando você publica o seu sistema, você pode se logar no seu sistema com o seu usuário do InteliForm.
    Porém existe uma excessão muito importante porque o seu sistema nunca é publicado com login via Facebook. Para você oferecer login via Facebook você precisa criar o seu sistema, se logar na interface administrativa e habilitar o login via Facebook. E isso só é possível após você criar o seu sistema e cadastrar o seu sistema como um aplicativo do Facebook, como explicado no item "Login via Facebook".
    Portanto se você se cadastrou no InteliForm via login do Facebook, você não vai conseguir se logar no seu sistema via login do Facebook porque ele não está habilitado quando o sistema é criado.
    Para você se logar no seu sistema, clique no menu superior em "Reinicializar Senha" e informe o seu usuário do Facebook que você utilizou para se cadastrar no InteliForm. Até esse momento, você não possui uma senha porque o InteliForm não armazena senha do Facebook. Esse simplesmente autoriza os seus usuários via a integração que existe entre o InteliForm e o Facebook. Portanto nesse caso quando você clica em "Reinicializar Senha" você estará criando uma conta comum para esse usuário. Será enviado um e-mail de confirmação que você receberá e deverá confirmar. Após a criação do seu usuário, no primeiro login você receberá outro e-mail de confirmação de senha que você também deverá confirmar. E pronto, você já pode se logar no seu sistema, e utilizá-lo normalmente e habilitar o login via Facebook.
    Se o seu usuário do InteliForm não foi criado pelo Facebook, você não precisará fazer nada disso porque o login é comum e não precisa ser habilitado. É automático.

  6. Login via Facebook
    O login via Facebook é uma facilidade oferecida pelo Facebook para os seus usuários e para os sistemas que são certificados e que seguem as regras de integração de sistemas.

  7. Login via Facebook no seu sistema
    Você pode incluir o login via Facebook no seu sistema. Esse procedimento leva aproximadamente 5 minutos para um profissional que conheça bem o processo de integração, mas pode ser complexo para quem não tem experiência com isso.
    Para publicar o seu aplicativo de cadastro no Facebook você precisa estar registrado como desenvolvedor no Facebook Developers, onde você deve prencher os dados solicitados com as suas preferências, e para as questõs abaixo, informar os seguintes dados:
    • App Domains: projetos.inteliform.com.br
    • Site com login do Facebook: http://projetos.inteliform.com.br//accounts/login/
    • URL: http://projetos.inteliform.com.br//
    • Secure Canvas URL: https://projetos.inteliform.com.br//
    • Page Tab URL: http://projetos.inteliform.com.br//
    • Secure Page Tab URL: https://projetos.inteliform.com.br//
    Anote o "App ID" e o "App Secret" do seu aplicativo gerado no Facebook Developer e cadastre-os na interface administrativa do seu sistema.
    Para adicionar o seu aplicativo na sua fanpage, ou para divulgar para outras pessoas instalarem o seu aplicativo nas suas páginas, basta seguir a instrução do item "Adding an App to a Page", disponível no link https://developers.facebook.com/docs/appsonfacebook/pagetabs/

  8. Se eu quiser o código do meu sistema gerado no InteliForm, eu posso adquiri-lo?
    Não, o nosso modelo de negócios utiliza o InteliForm como gerador de sistemas e a oferta do serviço de manutenção do seu sistema.

  9. Acesso aos dados do seu sistema
    Na interface administrativa você tem acesso aos dados que os seus usuários cadastram no seu sistema. E você pode fazer o download desses dados para backup ou para fazer análise em planilhas de texto.





Adicionais

Você pode contratar serviços adicionais para personalizar ainda mais o seu sistema.

  • Utilizar o domínio do seu portal no seu sistema InteliForm para ele ficar com o nome do seu domínio.
    Por exemplo: http://meusistema.minhaempresa.com.br
  • Utilizar certificado digital e criar um domínio seguro para o seu sistema (SSL).
    Por exemplo: https://meusistema.minhaempresa.com.br
  • Desenvolver um template para que o seu sistema utilize a mesma a identidade visual do seu portal.

Tudo isso é possível porque o seu sistema é completamente independente. Para a sua segurança, o seu sistema não compartilha nenhum tipo de dado com outros sistemas, e possui uma instância única de banco de dados para manter total isolamento e segurança das suas informações.
